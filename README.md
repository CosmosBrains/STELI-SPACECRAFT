## STELI SPACECRAFT

Welcome to this flappy bird inspired game, where you control a spaceship from E-Bay through some inconveniently placed pipes. Game was made in about 8-12 hours, please do not expect the videogame equivalent to the Mona Lisa, it's not happening.

The compiled code available under the MIT license, though **good luck** trying to use it since it's... compiled, AKA completely unreadable.

Anyway that's all I have to say, and the only reason it's public is so I can host it on GITHUB PAGES. Bye.
